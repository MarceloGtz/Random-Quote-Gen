## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceloGtz/Random-Quote-Gen.git
```

Go to the project directory

```bash
  cd Random-Quote-Gen
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
